# Zomato-Data-Analysis
Data Analysis 



This project explores and analyzes Zomato restaurant data to uncover insights into customer preferences, restaurant performance, and pricing trends.
The analysis provides a data-driven understanding of factors affecting restaurant ratings and helps in decision-making for both customers and business owners.

#Objectives

To clean, process, and visualize the Zomato dataset.

To identify key factors influencing restaurant ratings.

To analyze relationships between votes, cost, and online services.

To explore how restaurant features (like table booking and delivery options) affect ratings.



#Data Cleaning & Preparation

Removed missing or inconsistent rating entries.
Converted rate values to numeric for statistical analysis.
Checked and handled duplicate records.
Standardized column names for readability.
